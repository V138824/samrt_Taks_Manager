## Project Features
* **Add Tasks:** Quickly add new tasks to your list.
* **Local Storage:** Your tasks are saved in the browser so they don't disappear on refresh.
* **Interactive UI:** Clean and responsive design.
* **Task Management:** Easy to track and manage daily activities.

## Technologies Used
* **Frontend:** HTML5, CSS3, JavaScript (ES6)
* **Storage:** Browser LocalStorage
* **Version Control:** Git & GitHub

## How to Run
1. Clone the repository or download the zip file.
2. Open `src/index.html` in any modern web browser.
